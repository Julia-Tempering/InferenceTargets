Usage ‚Äã

To access the list of targetIds available in PosteriorDBTargets:

julia
ids = find_targetIds(PosteriorDBTargets)
length(ids)
147

To load one target:

julia
some_id = ids[1]
load_target(PosteriorDBTargets, some_id)
StanLogPotential(GLMM_Poisson_model_model)
